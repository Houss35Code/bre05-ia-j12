### Petit conseil de mentor :

Priorisez les ressources qui mentionnent les **champs privés** (utilisant le symbole `#`). C'est le standard actuel pour l'encapsulation en JavaScript Vanilla. Par exemple :

```javascript
class CompteBancaire {
  #solde = 0; // Propriété privée

  deposer(montant) {
    this.#solde += montant;
  }
}

```
